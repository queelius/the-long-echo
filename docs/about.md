## The Toolkit

**Data preservation:**

- **[CTK](https://github.com/queelius/ctk)** (Conversation Toolkit): AI conversations from all platforms, tree-aware storage, multi-format export
- **[BTK](https://github.com/queelius/btk)** (Bookmark Toolkit): Database-first bookmarks with hierarchical tags, content caching, NLP auto-tagging
- **[EBK](https://github.com/queelius/ebk)** (Ebook Toolkit): eBook metadata, full-text search, AI enrichment, semantic search
- **[PTK](https://github.com/queelius/ptk)** (Photo Toolkit): SHA256 dedup, AI annotation, multi-source import
- **[MTK](https://github.com/queelius/mtk)** (Mail Toolkit): Email archive with relationship mapping, notmuch integration
- **[JOT](https://github.com/queelius/jot)** (Journal): Plaintext idea journal with YAML frontmatter

**Orchestration:**

- **[longecho](https://github.com/queelius/longecho)**: Build static archives from toolkit exports
- **[longshade](https://github.com/queelius/longshade)**: Persona synthesis from personal data (spec only)

**Infrastructure:**

- **[repoindex](https://github.com/queelius/repoindex)**: Git repository catalog across 170+ repos
- **[crier](https://github.com/queelius/crier)**: Cross-post blog content to 12+ platforms
- **[src2md](https://github.com/queelius/src2md)**: Source code to Markdown for LLM consumption
